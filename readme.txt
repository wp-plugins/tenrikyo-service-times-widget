This is a widget that displays the current service times and time chart for the
Tenrikyo Church Headquarters service times.

== Description ==

This widget was built to display the current daily service times for the morning and evening services at the 
Tenrikyo Church Headquarters Main Sanctuary. At nearly every half a month, the service times change in sync with 
the times of sun rise and sun set.

= Features =
*	Automatically displays current service time.
*	Available in English and Japanese.
*	The title is customizable.

= Future Features =
*	Make translation files separate from the class file.
*	Make it easier to add events.

== Installation ==

1. Upload the `tenrikyo-service-times-widget.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. You will see the widget available under 'Appearance' -> 'Widgets'. Click and drag the widget into your desired 
position on the side-bar.
